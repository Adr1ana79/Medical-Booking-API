# SEng-Coursework
The system maintains a database of doctors, patients and appointments

В тази курсова работа се проектира, създава, тества, документира и защитава система за записване на посещения при лични лекари. Със системата се взаимодейства чрез приложно-програмен интерфейс (API) по HTTP(S) протокола.
